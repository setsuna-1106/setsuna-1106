08
Deploy

Server ready structure

当前版本可以静态部署；如果以后接服务器，只需要让数据层从 API 获取内容，项目筛选、任务统计和矩阵渲染不需要重写。

01 Static first

上传整个 advance 文件夹到 Nginx、GitHub Pages、Cloudflare Pages 或任意静态服务器。

02 Data layer

项目、任务、笔记、矩阵集中在 data.js，后续可替换成 /api/profile.json。

03 Client state

任务看板使用 localStorage 保存本地编辑，服务器版可改为数据库同步。

python3 -m http.server 8080 --directory advance
