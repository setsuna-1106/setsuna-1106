04
Workflow

C computes. Python explains.

仓库采用清晰的计算工作流：C 负责性能关键的数值循环，把数据写成 CSV 或二进制；Python 负责探索、检查、画图与论文式输出。部分实时项目直接使用 Raylib 在 C 端完成渲染。

01 Model

把物理问题写成方程、边界条件和初始条件。

02 Compute

用 C 编写数值核心，控制步长、误差和输出格式。

03 Visualize

用 Python 或 Raylib 检查轨迹、分布、相图和误差。

04 Document

把推导、代码入口和观察结果放回笔记。
